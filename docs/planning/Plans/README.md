# Plan Mode notes

Implementation plans from Cursor or Claude Plan Mode live here. Each file is keyed by the same **PRD ID** as the PRD it supports.

**Naming:** `PRD-YYYY-NNN__cursor-plan.md` or `PRD-YYYY-NNN__claude-plan.md`

**Linking:** The PRD’s YAML frontmatter includes `plans.cursor` and `plans.claude` paths pointing to these files. The [ROADMAP](../ROADMAP.md) can link to them from the “Now” section.

No plans here yet. When you create a plan file for a PRD, add it and reference it from the PRD and roadmap.
