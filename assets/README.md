# Phína assets

Add the following image files for app icon and splash:

- **icon.png** — 1024×1024 app icon
- **splash-icon.png** — Splash screen image (centered)
- **adaptive-icon.png** — 1024×1024 Android adaptive icon foreground
- **favicon.png** — Web favicon

Until these exist, you may need to temporarily remove or change the `icon` and `splash` paths in `app.config.ts` to run the app.
